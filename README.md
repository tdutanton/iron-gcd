# Калькулятор НОД (GCD Calculator)

Это **учебный проект** на языке Rust, демонстрирующий создание простого веб-сервера с использованием фреймворка **Iron**. Сервер принимает два (или более) целых положительных числа через HTML-форму и вычисляет их **наибольший общий делитель (НОД / GCD)**.

draft for o'reilly rust book

## Возможности

- Отображение HTML-формы для ввода чисел.
- Приём данных через POST-запрос.
- Парсинг и валидация входных чисел.
- Вычисление НОД для произвольного количества чисел.
- Отображение результата в веб-браузере.

## Используемые технологии

- **Rust** — системный язык программирования.
- **Iron** — веб-фреймворк (устаревший, но удобен для обучения).
- **router** — маршрутизация запросов.
- **urlencoded** — парсинг формы с `application/x-www-form-urlencoded` данными.

## Запуск проекта

1. Убедитесь, что у вас установлен [Rust и Cargo](https://www.rust-lang.org/tools/install).
2. Клонируйте или распакуйте проект.
3. В корне проекта выполните:

   ```bash
   cargo run
   ```

4. Откройте в браузере: http://localhost:3000
